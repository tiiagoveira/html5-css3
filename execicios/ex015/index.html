Utilizando Externos

Está parte de CSS é feita na folha de estilo (externa)

                @charset "UTF-8";                                   * informando que pode ter caracteres especiais

                body {
                    background-color: lightskyblue;                 * cor de fundo
                    font-family: Arial, Helvetica, sans-serif;      * tipo da fonte
                    font-size: 20px;                                * tamanho da fonte em (px)
                }

                h1 {
                    color: darkblue;                                * cor da fonte
                    background-color: lightblue;                    * cor de fundo da fonte
                    font-size: 1.5em;                               * tamanho da fonte em (em)
                }

                h2 {
                    color: darkred;                                 * cor da fonte
                    font-size: 1.2em;                               * tamanho da fonte em (em)
                }

                p {
                    text-align: justify;                            * alinhamento da fonte
                }
