Listas ordenadas

<ol type=" " start=" "> - utlizado para criar listas ordenadas.

<li> </li> - acrescenta item a lista.

            type - tipo de lista
                1 - lista numérica
                a - lista alfabética minúscula
                A - lista alfabética maiúscula
                i - lista números romanos minusculo
                I - lista números romanos maiusculo
            
            start - por onde começar na lista, sempre infomado por um número 
                3 - a lista irá iniciar da letra (c)
