Utilizando Estilos Inline

                <body style="background-color: lightskyblue; font-family: Arial, Helvetica, sans-serif; font-size: 10px;">
                     * style: abertura para as tags CSS
                     * background-color: cor do fundo da página
                     * font-family: tipo de fonte
                     * font-size: tamanho da fonte

Formatação no próprio (h1) - primeiro

                <h1 style="color: darkblue; background-color: lightblue; font-size: 1.5em;"> </h1>
                    * color: cor da fonte (do h1)
                    * background-color: cor de fundo da fonte (atrás)
