Usando links externos

<a href=" " target=" " rel=" "> - utilizado para adicionar links 
        href - url ou caminho da página a ser exibida
        
        target - como esse link irá abrir
            "_blank" - abrir em uma nova aba
            "_parent" - abre o link no frame do elemento pai
            "_self" - abre na mesma página
            "_top" - abre um link vinculado em todo o corpo da janela
        
        rel - relação entre a página de origem e o destino do link
            "external" - externo
            "next" - proxima página
            "prev" - página anterior
            "nofollow" - indicar aos mecanismos de busca que um link não deve ser seguido

quando se quer voltar para a página inicial e a página atual está em uma pasta diferente utiliza-se no (href) o ( . . / ) mais o nome da página
