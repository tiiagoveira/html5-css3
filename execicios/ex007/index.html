Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando usando a tag <b></b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong></strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i></i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em></em> (semântica).

Marcação / Destaque

Podemos criar também um texto marcado usando a tag <mark></mark> (semântica).

Texto grande e pequeno

Estamos criando um texto grande utilizando a tag <big></big> (não semântica) e um texto pequeno nesse parágrafo utilizando a tag <small></small> (semântica)

Texto deletado

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado utilizando a tag <del></del> (semântica).

Texto inserido

Existe também o sublinhado com a tag <u></u> (não semântica).

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois utilizando a tag <ins></ins> (semântica).

Texto sobreescrito

Para inserir coisa do tipo X20 utilizando a tag <sup></sup> (semântica)

Texto subscrito

Para inserir coisas do tipo H2 utilizando a tag <sub></sub> (semântica)
