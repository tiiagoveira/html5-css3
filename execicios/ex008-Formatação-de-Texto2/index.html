Código-fonte

utilizamos o code para organizar as letrar deixando o espaçamento de todas com a mesma largura

assim não fica o código de uma forma desistruturada.


System.out.println("Hello World") - sem o code

System.out.println("Hello World") - com o code

Pré-formatação

utilizamos o pre para respeitar os espaços do inicio da linha de comando estabelecido no codigo digitado dentro da IDE


Sem o pre

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else print(f'O número {num} é IMPAR') print('Fim do programa')


Com o pre

 
            
                num = int(input('Digite um número'))
                if num % 2 == 0:
                    print(f'O número {num} é PAR')
                else
                    print(f'O número {num} é IMPAR')
                print('Fim do programa')

Citações simples

Como diria Fulano: Água mole pedra dura, tanto bate até que fura.

utiliza blockquote para citações completas, deslocando o texto para a direita com um parágrafo.
